# CET-4 智能考试管理平台

基于 Vue 3 + FastAPI + MySQL 的 CET-4 英语四级智能考试系统，支持题库管理、AI 难度分析、智能组卷、在线考试与成绩分析。

## ✨ 功能模块

| 模块 | 说明 |
|------|------|
| **题库管理** | PDF 真题自动解析上传、题目 CRUD、批量导入导出、音频自动匹配 |
| **难度分析** | textstat 静态分析 + DeepSeek AI 评估 + Elo 动态难度校准，三层递进式评估体系 |
| **智能组卷** | 按 CET-4 官方 57 题标准结构，基于难度权重矩阵自动生成标准化试卷 |
| **在线考试** | 客观题自动判分 + 主观题 AI 智能评分 |
| **成绩分析** | 学生成绩分布、题目难度统计等多维度分析报告 |

## 🏗️ 技术栈

### 后端
- **框架**: FastAPI (Python 异步 Web 框架)
- **ORM**: SQLAlchemy 2.0
- **数据库**: MySQL 8.0
- **AI**: DeepSeek API（难度评估 + 主观题评分）
- **PDF 解析**: pdfplumber / pypdfium2 + EasyOCR
- **文本分析**: textstat（可读性评估）

### 前端
- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI 组件库**: Element Plus
- **状态管理**: Pinia
- **图表**: ECharts
- **路由**: Vue Router

## 🧠 智能组卷算法

组卷模块是整个系统的核心，采用**基于多属性约束的分层加权随机抽题算法**，融合四项关键技术：

### 难度权重矩阵

根据目标难度等级（1-5），从各难度桶中按比例抽题：

| 目标难度 | 难度1 | 难度2 | 难度3 | 难度4 | 难度5 |
|---------|------|------|------|------|------|
| 1 (基础) | 45% | 30% | 15% | 7%  | 3%  |
| 2 (进阶) | 25% | 35% | 25% | 10% | 5%  |
| 3 (中等) | 10% | 20% | 40% | 20% | 10% |
| 4 (较难) | 5%  | 10% | 25% | 35% | 25% |
| 5 (困难) | 3%  | 7%  | 15% | 30% | 45% |

### Passage 块级聚合

对于阅读类题型（选词填空、段落匹配、仔细阅读），以 Passage 为单位进行题目分组抽取，确保同一篇文章下的题目保持上下文完整性，杜绝"孤题"现象。

### 内容相似度去重

基于 `difflib.SequenceMatcher` 进行文本相似度比对（阈值 0.85），避免试卷中出现高度相似的重复题目。

### CET-4 标准卷面结构

严格按照 CET-4 官方规范生成 57 题固定框架：

| 题号 | 题型 | 题数 | 分值 |
|------|------|------|------|
| 1 | 写作 (Part I) | 1 | 106.5 |
| 2-26 | 听力 (Part II) | 25 | 248.5 |
| 27-36 | 选词填空 (Section A) | 10 | 35.5 |
| 37-46 | 段落匹配 (Section B) | 10 | 71.0 |
| 47-56 | 仔细阅读 (Section C) | 10 | 142.0 |
| 57 | 翻译 (Part IV) | 1 | 106.5 |
| **合计** | | **57** | **710** |

## 🚀 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- MySQL 8.0+

### 后端配置

```bash
cd exam/exam-backend

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入数据库连接信息和 DeepSeek API Key

# 初始化数据库表
python init_db.py

# 启动后端服务
python main.py
# 或: uvicorn main:app --reload --port 8000
```

### 前端配置

```bash
cd exam/exam-frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

### API 文档

后端启动后访问 http://localhost:8000/docs 查看 Swagger 自动生成的 API 文档。
